Questa informativa integra la Privacy Policy.

1. Tecnologie di tracciamento usate

L’App può utilizzare:

  • identificatori del dispositivo
  • identificatori pubblicitari (IDFA / Android Ad ID)
  • SDK terze parti (Google AdMob, Firebase Analytics, Crashlytics)
  • token notifiche push

2. Servizi di terze parti

2.1 Google AdMob – Pubblicità

Annunci personalizzati o non personalizzati, secondo preferenze utente.

Dati trattati: ID pubblicitario, dati dispositivo, interazione con annunci.

Base giuridica annunci personalizzati: consenso.

2.2 Firebase Analytics – Statistiche

Dati anonimi per migliorare l’app.

Base giuridica: legittimo interesse o consenso se richiesto.

2.3 Firebase Crashlytics – Crash report

Informazioni tecniche sugli errori per stabilità app.

3. Consenso utente

Al primo utilizzo o tramite impostazioni l'utente può scegliere tra:

  • accettare tutto
  • rifiutare tracking non essenziali
  • selezionare singole preferenze

Fino a consenso, AdMob personalizzato NON viene attivato.

4. Disattivazione ID pubblicitari

  • Android: Impostazioni → Privacy → Annunci
  • iOS: Impostazioni → Privacy → Pubblicità

Dati trattati da Google Ireland Limited / Google LLC con SCC GDPR.
